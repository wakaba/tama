「たまてばこ」とは

各サイトの更新時刻を取得してHTMLに出力する、 いわゆる「アンテナ」と呼ばれるシステムです。同様のシステムには 朝日奈アンテナ、 なつみかん、 WDBなどがあります。

次のような特徴を持っています。

  • Rubyで書かれています。
  • 他のアンテナの出力する形式に対応しています(LIRS,DI,hina.txtに対応)。
  • インストーラが付いています。
  • 設定がシンプルです。

一方、次のような制限があります。

  • Unix系OS以外の環境は考えていません。
  • 複数同時接続には対応していません。
  • 認証が必要なサイトの更新時刻は取得できません。
  • 機能が少ないです:-)

「たまてばこ」の動作例としては、 素子アンテナがあります。

ダウンロード

最新のバージョンは、安定版1.0.0(2001/01/30リリース)、 開発版1.1.63(2001/06/18リリース)です。

  • tama-1.0.0.tar.gz
  • 開発版＆古いバージョン

設置にはruby ver 1.4.6もしくは1.6.xが必要になります。他のバージョンでも動くかもしれません。

開発環境はKondara MNU/Linux(Mary-RC2) + ruby-1.6.3 and ruby-1.4.6、 動作確認環境はこれに加えて、FreeBSD 4.1.1-RELEASE + ruby-1.6.1 and ruby-1.4.6です。

インストール

ruby install.rbとすると、インストーラが起動します。 以下の点について聞かれますので、それぞれ答えて下さい。

  • rubyのパス
  • gzipのパス
  • 設定ファイルをインストールするディレクトリ
  • 出力先ディレクトリ
  • アンテナのURL

アップデート時も同様にインストーラを使って下さい。 設定ファイルは上書きされませんが、スクリプトは上書きされます。 念のため、バックアップを取っておく事をお勧めします。

この状態でインストールしたディレクトリに移動し、 ruby tama.rbとすると動作します。 設定方法は、設定リファレンスをご覧下さい。

  • 起動オプション
    • --version : バージョンを表示する。
    • --noget : 取得を行わずにHTML出力だけする。
    • --debug : 取得状況を日本語で表示する。
    • --help : 詳しい使い方を表示する。
    • --check : 設定ファイルをチェックして終了する。
    • --local : ローカルファイル(チェックするURLがfile://のもの)のみ更新する。

セキュリティ

rubyのセキュリティチェック機能を利用しています。ですが、 プログラムの性質上、以下の内容は信頼できると仮定しています。

  • conf/ ディレクトリ以下にある各種設定ファイル
  • html/ ディレクトリ以下にある出力元のHTMLファイル
  • 起動オプション

これらのファイルを勝手に変更されないように気を付けて下さい。
